# PyGame Test

## Controls

### Yellow Player
```
W = UP 
A = LEFT
S = DOWN
D = RIGHT
LCTRL = FIRE BULLET
```
### Red Player
```
I = UP
J = LEFT
K = DOWN
L = RIGHT
RCTRL = FIRE BULLET
```
## Requirements

Requirements:

- Python 3.9
- PyGame 2.1.0


How to install:
```
pip3 install pygame

```
or
```
pip install pygame
```
